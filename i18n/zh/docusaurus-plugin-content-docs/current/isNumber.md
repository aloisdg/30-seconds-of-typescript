检查给定的参数是否是数字。

Use `typeof` 检查值是否分类为数字原始。
保护反对 `NaN`, 检查是否 `val === val` (as `NaN` has a `typeof`等于 `number` 并且是唯一不等于自身的值）。

```ts title="typescript"
const isNumber = (val: any) => typeof val === "number" && val === val;
```

```ts title="typescript"
isNumber(1); // true
isNumber("1"); // false
isNumber(NaN); // false
```
