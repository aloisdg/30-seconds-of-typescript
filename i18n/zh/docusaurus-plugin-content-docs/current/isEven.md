[用中文] 
返回 `true` 如果给定的数字甚至是 `false` 否则.

检查一个数字是奇数还是使用模量(`%`) operator.
返回 `true` 如果数字是 even, `false` 如果数字是 odd.

```js
const isEven = (num) => num % 2 === 0;
```

```js
isEven(3); // false
```
